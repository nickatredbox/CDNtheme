This theme has been tested in IE8, Chrome & Firefox 12.0

The Theme Folder

Here is the folder structure under which all the objects used in this template are located

All the content and functionality you can see on this page is contained in this structure. This page can be used as a templete for generating content in a standard theme.

Conten specifically used in this template and not required for use of the theme is located in the root of theme. This includes images and markup pages like this one.

The Images Folder

All the images used in the theme are in this location

The Markup Folder

The markup folder contains the following files, These files are partial HTML pages containing relevent markup for the specific area of the page as well as references to any style sheets required

The Scripts Folder

The Javascript is located in the script folder. Patrial pages like DefualtNav.htm are delivered to the containg page using the Javascript in this folder.

The Javascript in LincolnDefault.js is used to fetch the content in the partial pages and render them inside the container page.

The Styles Folder

The styles folder has all the CSS files needed for the theme divided into seperate files, each concerned with a specific part of the page or partial page.

DefaultNav.htm shown below shows how the style sheet nav.css is used
